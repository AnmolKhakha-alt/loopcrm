# Premium SaaS UI Skill

Always design UI with:

* modern premium SaaS style
* dark theme
* electric blue accent colors
* glassmorphism cards
* clean typography
* spacious layouts
* responsive mobile-first design
* smooth Framer Motion animations
* professional dashboard aesthetics inspired by Stripe, Linear, and Vercel

UI Requirements:

* avoid clutter
* use consistent spacing
* rounded-xl or rounded-2xl components
* subtle hover effects
* soft shadows
* loading skeletons
* empty states
* polished transitions

Component Standards:

* reusable components
* shadcn/ui patterns
* accessible UI
* clean sidebar layouts
* sticky headers where useful

Never generate outdated or generic dashboard designs.
Always prioritize premium professional appearance.
