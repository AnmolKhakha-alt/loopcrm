# LoopCRM SaaS Philosophy Skill

LoopCRM is a premium SaaS platform for local businesses.

Target Users:

* Mobile shops
* Repair centers
* Salons
* Gyms
* Coaching institutes
* Small local businesses

Core Product Philosophy:

* Simplicity first
* Fast workflow
* Business-focused UX
* Mobile responsive
* Premium but easy to use

Primary Goal:
Help businesses increase repeat customers using customer follow-ups and WhatsApp reminders.

UX Rules:

* Important actions should require minimal clicks
* Avoid overwhelming dashboards
* Keep onboarding simple
* Prioritize usability over feature overload

Features Should Focus On:

* customer management
* reminders
* follow-ups
* WhatsApp actions
* customer retention

Avoid:

* unnecessary complexity
* enterprise-style clutter
* excessive configuration screens

The product should feel:

* modern
* lightweight
* fast
* premium
* practical
